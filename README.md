# AI Interior Design Generator

A Streamlit app that generates photorealistic interior design images from a
floor plan + a set of customizable requirements (room, style, wall material,
flooring, windows, doors, lighting, color palette). Images are generated by
calling a text-to-image model on the **Hugging Face Inference API**.

## How it works
1. You (optionally) upload a house floor plan image — it's shown as a visual
   reference only; the app doesn't analyze it with computer vision.
2. You pick a room and design parameters from dropdowns.
3. The app builds a detailed text prompt from your choices.
4. The prompt is sent to a Hugging Face model (FLUX.1-dev, SDXL, or SD 2.1)
   via `POST https://api-inference.huggingface.co/models/{model_id}`.
5. The returned image is displayed and downloadable.

## 1. Get a Hugging Face API token
1. Create a free account at https://huggingface.co
2. Go to https://huggingface.co/settings/tokens and create a token
   ("Read" access is sufficient for inference).
3. For gated models like `black-forest-labs/FLUX.1-dev`, open the model page
   and click "Agree and access repository" first, or the API will return a
   403 error even with a valid token.

## 2. Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
Paste your token into the sidebar field, or save it permanently:
```bash
mkdir -p .streamlit
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
# edit .streamlit/secrets.toml and paste your real token
```

## 3. Deploy on Streamlit Community Cloud
1. Push this folder to a GitHub repo.
2. Go to https://share.streamlit.io -> "New app" -> pick the repo/branch and
   set the main file to `app.py`.
3. In the app's **Settings -> Secrets**, add:
   ```toml
   HF_API_TOKEN = "hf_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
   ```
4. Deploy. The token will be picked up automatically as the sidebar default.

## Notes on model choice
- `black-forest-labs/FLUX.1-dev` — best image quality, gated (requires
  accepting license on the model page), slower and may need a paid
  Inference Endpoint for reliable uptime.
- `stabilityai/stable-diffusion-xl-base-1.0` — good quality/speed balance,
  ungated.
- `stabilityai/stable-diffusion-2-1` — fastest, lower quality, good for
  quick iteration.

The free Inference API has rate limits and models "cold start" (can take
20-60s the first time you call them — the app automatically retries while
a 503 "loading" response is returned).

## Files
